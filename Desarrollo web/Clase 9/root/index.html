Git es un sistema de control de versiones gratuito y de codigo abierto, diseñado para manejar desde pequeños a grandes proyectos de manera rapida y eficaz. Se entiende como control de versiones a todas las herramientas que nos permiten hacer modificaciones en nuestro proyecto. Este sistema registra los cambios realziados sobre un archivo o conjunto. git config user.name -> Devuelve nombre de usuario git config --global user.name "NombreUsuario" -> Asignamos el nombre de usuario. git config --global user-email "Email" -> Asignamos el email. git init -> Inicializa un repositorio git status -> Devuelve un estado del estado git add -> agregamos los archivos listos git commit -> Guardamos cambios y dejamos comentario git branch "nombre" -> Creea una rama git branch -l -> Listamos las ramas git checkout "nombreRama" -> Cambiamos a la rama que creamos git log --oneline -> lista los commits git merge "NombreRama" -> Une la rama creada a la master git branch -D "NombreRama" -> elimina la rama GIT: LOS 3 ESTADOS 1 - (el que trabajamos) "Preparamos las cajas" - working directory git add 2 - (archivos listos) "agregamos las cajas listas - staging area" git commit 3 - (registro de todos los archivos) "lote listo" - repository
